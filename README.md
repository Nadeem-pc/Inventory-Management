## 📦 Inventory Management (TypeScript)  
A simple inventory management system built with **TypeScript**, allowing users to:  
- Add new products  
- Edit existing products  
- Delete products  
- List all products  

---

## 🚀 Features  
- ✅ Add new products  
- ✅ Edit existing products  
- ✅ Delete products  
- ✅ List all products  

---


🛠️ Technologies Used
  * TypeScript
  * Node.js
  * Express.js
  * MongoDB
